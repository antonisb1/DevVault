# DevVault Frontend

React + Vite frontend for DevVault.

## Local setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Notes

- Uses React Query for server state
- Uses Zustand for auth and UI state
- Uses Tailwind CSS v4 with the official Vite plugin
- Uses Ant Design selectively for tables, modals, buttons, and theme integration
